# 🎙️ Text-to-Audio AI Generator

A powerful AI-based Text-to-Speech (TTS) web application built with Python and Flask. This project converts text into natural-sounding audio using the Kokoro ONNX model and provides a simple web interface for generating and downloading speech in WAV format.

## 🚀 Features

- 🧠 AI-powered text-to-speech generation
- 🌐 Web interface built with Flask
- 🔊 Generates high-quality `.wav` audio files
- ⚡ Fast inference using ONNX Runtime
- 📦 Lightweight and easy to deploy
- 📝 Supports custom text input
- 💾 Download generated audio instantly

## 🛠️ Technologies Used

- Python 3.x
- Flask
- ONNX Runtime
- NumPy
- SoundFile

## 📂 Project Files

- `app.py` — Flask web application
- `kokoro_tool.py` — Text-to-speech generation logic
- `requirements.txt` — Python dependencies
- `.gitignore` — Excluded files and folders
- `web_output.wav` — Example generated output
- `README.md` — Project documentation
- `LICENSE` — MIT License
